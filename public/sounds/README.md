# Suoni per Inferno Console

Questa cartella contiene i file audio per gli effetti sonori di Inferno Console.

## File Richiesti

Per avere suoni di qualità, scarica i seguenti file e mettili in questa cartella:

### 1. Applausi (applause.mp3)
- **Fonte consigliata**: [Pixabay - Applausi](https://pixabay.com/it/sound-effects/search/applausi/)
- **Fonte alternativa**: [SoundDino - Applausi](https://sounddino.com/it/effects/applause/)
- **Descrizione**: Applausi realistici di una folla
- **Durata consigliata**: 2-4 secondi
- **Formato**: MP3, WAV o OGG

### 2. Trombetta (horn.mp3)
- **Fonte consigliata**: [Myinstants - DJ Airhorn](https://www.myinstants.com/it/instant/dj-airhorn/)
- **Fonte alternativa**: [Depositphotos - Trombe](https://depositphotos.com/it/sound-effects/spettacolo.html)
- **Descrizione**: Suono di trombetta da DJ
- **Durata consigliata**: 1-3 secondi
- **Formato**: MP3, WAV o OGG

### 3. Effetti di Transizione
- **swoosh.mp3**: Effetto di movimento rapido
- **whoosh.mp3**: Effetto di vento
- **drop.mp3**: Effetto di caduta
- **rise.mp3**: Effetto di ascesa

### 4. Effetti Comici
- **beep.mp3**: Suono di notifica
- **squeak.mp3**: Suono che perde

## Come Scaricare

1. Vai sui siti consigliati
2. Cerca i suoni appropriati
3. Scarica i file in formato MP3, WAV o OGG
4. Rinomina i file secondo la lista sopra
5. Metti i file in questa cartella (`public/sounds/`)

## Fallback

Se i file non sono presenti, Inferno Console userà automaticamente i suoni sintetici generati dal sistema.

## Licenze

Assicurati che i suoni scaricati siano:
- **Royalty-free** (senza diritti d'autore)
- **Compatibili** con l'uso commerciale
- **Di buona qualità** (almeno 44.1kHz, 16-bit)

## Note

- I file devono essere in formato MP3, WAV o OGG
- La dimensione consigliata è sotto i 500KB per file
- I suoni devono essere chiari e senza distorsioni
